# Proyecto Intermodular – Gestión de Reservas

## Descripción
Aplicación web para la gestión de reservas de recursos en un centro educativo,
permitiendo a distintos tipos de usuarios consultar y administrar disponibilidad.

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
- Git y GitHub
