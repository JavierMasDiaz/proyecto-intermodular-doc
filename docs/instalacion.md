# Instalación

## Requisitos
- Servidor web Apache
- PHP 8.0 o superior
- MySQL
- Navegador web moderno

## Pasos de instalación
1. Clonar el repositorio
2. Importar la base de datos
3. Configurar el archivo `.env`
4. Acceder a la aplicación desde el navegador

## Variables de entorno
- DB_HOST
- DB_USER
- DB_PASSWORD
- DB_NAME
